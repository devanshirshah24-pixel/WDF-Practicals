# StudentHub Portal

## Project Description

StudentHub Portal is a simple web-based application designed to help students access important academic information from a single platform. The portal provides features such as course management, attendance tracking, assignment viewing, result checking, and account settings.

The main objective of this project is to create a user-friendly interface where students can easily navigate between different academic services without visiting multiple systems. The portal includes separate pages for Login, Sign Up, About Us, Courses, Attendance, Assignments, Results, and Settings.
